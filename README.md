# Zephyr Card Starter

A premium motion-led Next.js starter website for Zephyr Card.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Motion for React
- lucide-react icons

## Local setup

```bash
npm install
npm run dev
```

## Routes

- `/` homepage
- `/partner` partner interest form
- `/thanks` confirmation page

## Notes

The partner form currently uses a placeholder Formspree endpoint. Replace the form `action` in `app/partner/page.tsx` with your real endpoint or wire it to Supabase.
